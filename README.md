## **HTML5 Lab Exercise: Create a Semantic Web Page**

### **Lab Title**

**Design a Semantic HTML5 Webpage Using HTML5 Semantic Elements**

### **Objective**

Create a simple webpage using **HTML5 semantic tags** to organize the content in a meaningful and structured way.

### **Problem Statement**

Create an HTML5 webpage with the title **"My Website"** using appropriate **semantic HTML5 elements**.

Your webpage should include the following:

1. A **header** section containing:

   * Website title: **My Website**
   * A short description:

     > Welcome to my semantic HTML5 example

2. A **navigation menu** containing the following links:

   * Home
   * About
   * Gallery
   * Contact

3. An **About Us** section containing:

   * Heading: **About Us**
   * Paragraph:

     > This section describes the purpose of the website and provides background information.

4. A **Gallery** section containing:

   * Heading: **Gallery**
   * Two images:

     * Beautiful landscape
     * City skyline
   * Each image should have appropriate **alternative text**.
   * Add the following caption below the second image:

     > The city skyline captured at dusk.

5. A **Contact** section containing:

   * Heading: **Contact**
   * The following text:

     > You can reach us via email at [info@example.com](mailto:info@example.com).

6. A **footer** section displaying:

   * © 2026 My Website. All rights reserved.

### **Requirements**

* Use the proper HTML5 document structure.
* Use the following semantic elements wherever appropriate:

  * `<header>`
  * `<nav>`
  * `<main>`
  * `<section>`
  * `<figure>`
  * `<figcaption>`
  * `<footer>`
* Use heading tags (`<h1>`, `<h2>`), paragraphs (`<p>`), images (`<img>`), and hyperlinks (`<a>`).
* Ensure the webpage is well-structured and easy to read.

### **Learning Outcomes**

After completing this lab, students will be able to:

* Understand the purpose of HTML5 semantic elements.
* Build a structured webpage using semantic tags.
* Organize webpage content into logical sections.
* Improve webpage readability and maintainability using semantic HTML.
